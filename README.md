## 🚀 Quick Start

```bash
cd pureons
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📂 Project Location

The full project lives in [`pureons/`](pureons/). See the [README](pureons/README.md) inside for complete documentation.

## ✨ Highlights

- **Light & Dark theme** with persistent toggle
- **Dynamic filtering** by category (no reload)
- **Smooth animations** via Framer Motion
- **Responsive** — mobile to desktop
- **Clean component architecture** — easy to extend
